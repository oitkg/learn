# 01.python语言介绍和特点

## python特点

1.简洁、优雅、简单

2.面向对象的语言

3.解释性语言（需要解释器来按行来执行）

4.高级语言

5.扩展性非常好

6.第三方支持非常丰富

7.容易和其他语言集成

## Python可以为那些人提供帮助

1.想学语言的孩子（启蒙语言）

2。自动化工程师

3.人工智能工程师

4.大数据工程师
