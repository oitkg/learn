# 03.语言基础

## 变量

1.变量名是一种标识符

2.变量名、函数名、方法名、类名由字母、数字、下划线组成，不能以字母开头不能使用保留字，不使用内置函数名

```python
x = 100 
y = x
x = 'abc'
输出结果为：x = abc  y = 100
```

100、x为值类型，变量存在栈里，100、x存在常量（值）里，并进行对应，没有使用堆。

```python
x = ['a','b','c']
y = x
x[0] = 100
输出结果为：x = ['100','b','c']  y = ['100','b','c']
```

['a','b','c']、['100','b','c']为引用类型，变量存在栈里，'a'、'b'、'c'、100存在常量（值）里，列表在堆空间里创建，并进行对应。
